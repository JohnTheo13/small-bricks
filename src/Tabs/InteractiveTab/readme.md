## PointingTab Component

Quote tab for web-app

### PropTypes

| propName | propType | defaultValue | isRequired | options/example |
|----------|----------|--------------|------------|---------|
| onClick  | function | -            | +          |  |
| children   | any   | -            | +          |  |
| active   | bool   | -            | +          |  |

### Example

``` js
<InteractiveTab active={false} onClick={props.onClick} >
  Hello Joey
</InteractiveTab>
```
