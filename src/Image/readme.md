## Image Component

Image components, used for specific size icons-images

### PropTypes

| propName | propType | defaultValue | isRequired | options/example |
|----------|----------|--------------|------------|---------|
| onClick     | func   | -      | -          |  |
| tag   | string    | -          | +           | 'small','medium','round'|
| src   | string    | -          | +           | src |
| style | object    | -          | -           |     |

### Example

``` js
<Image src="images/eve.png" tag='small' />
```
